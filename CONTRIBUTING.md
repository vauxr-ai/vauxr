# Contributing to Vauxr Local

Thanks for your interest in contributing! Here's what you need to know.

## Contributor License Agreement

Before your pull request can be merged, you'll need to sign the [Contributor License Agreement](CLA.md). This is handled automatically by [CLA Assistant](https://cla-assistant.io) — you'll be prompted when you open your first PR. It's a one-time step.

The CLA uses a **license grant** model: you keep your copyright, but grant the maintainer the right to use your contributions under any license (including commercial). This is what makes the open-core model sustainable.

## Getting Started

- Fork the repo and branch off `develop`
- `main` = stable releases, `develop` = integration branch
- Open PRs targeting `develop`

## Code Style

- Follow the conventions you see in the existing code
- Keep commits focused and atomic
- Write a clear commit message

## Reporting Issues

Open a GitHub Issue. Include steps to reproduce, expected vs actual behavior, and your environment.

## Questions

Open an issue or start a discussion. We're friendly.
